Cyber4j
=======
Java and android client for CyberWay blockchain.

Download
--------
1. add `maven { url 'https://jitpack.io' }` to your repositories clause
2. add `implementation 'com.github.GolosChain:cyber-android:0.71' in dependencies clause`
